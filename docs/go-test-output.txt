== Gated hardware run ==

The explicit hardware gate is a manual macOS RDMA validation path, not part of
ordinary test runs. Discovery/failure-path checks require `JACCL_TEST_RDMA=1`.
Tests that transition queue pairs to RTR also require
`JACCL_TEST_RDMA_ALLOW_RTR=1`, including direct `TestIntegrationChild` runs.
Local loopback RTR is rejected unless `JACCL_TEST_RDMA_ALLOW_LOCAL_LOOPBACK=1`
is also set; the normal physical path is one `TestIntegrationChild` process per
host.
